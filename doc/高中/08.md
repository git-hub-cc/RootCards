**词根（通假规律）：u, v, w 相互通假**

**单词1：wine / vine / vinegar**
*   **wine** （葡萄）酒
*   **vine** 葡萄藤
*   **vinegar** 醋

**例句1：**
*   Grapes on the vine. (葡萄藤上的葡萄。)
*   grape vinegar (葡萄醋)

**补充笔记1：**
*   `wine` (酒) 和 `vine` (葡萄藤) 是同源词，因为葡萄酒是由葡萄藤上的果实酿造的。
*   `wine` 和 `vinegar` (醋) 也是同源词，它们的酿造过程（发酵）相似，只是具体方法和产物不同。

**词根：de- (远离；向下)**

**单词2：defect** 缺点，瑕疵

**词根：vinc- / vict- (赢)**

**单词1：win / victory / invincible / convince**
*   **win** 赢
*   **victory** 胜利
*   **invincible** 不可战胜的
*   **convince** 说服，使信服

**补充笔记1：**
*   `win` 通过 `w/v` 通假演变为 `vinc-`。
*   `victory` 是 `win` 的同源名词。
*   `invincible` = `in-` (不) + `vinc-` (赢) + `-ible` (能...的) = 不能被赢的。
*   `convince` = `con-` (全部) + `vinc-` (赢) = 把对方全部赢了，即说服。

**词根（B字母的象形特征：凸起，鼓出）**

**单词1：ball / balloon / belly / belt**
*   **ball** 球
*   **balloon** 气球
*   **belly** 肚子
*   **belt** 腰带；安全带

**词根：break (打破)**

**单词1：break / fragile / fragment / fragrant**
*   **break** 打破；休息
*   **fragile** 易碎的
*   **fragment** 碎片
*   **fragrant** 香的

**补充笔记1：**
*   `break` 的核心意境是“断开”。
*   `fragile` = `frag-` (break的变体) + `-ile` (能够...的) = 能够被打破的。
*   `fragment` 是打破后形成的东西。
*   `fragrant` 是打破果实或花朵后闻到的香味。

**单词2：brake** 刹车
**补充笔记2：** `brake` 是 `break` 的同源词，刹车就是“断开”车辆的动力系统。

**词根：bind / band (绑)**

**单词1：bind / bound**
*   **bind** 绑 (动词，过去式/分词 bound)
*   **bound** 绑 (作为独立动词)

**单词2：bond / bunch / boundary / border**
*   **bond** 纽带
*   **bunch** 一束，一捧
*   **boundary** 分界线
*   **border** 边境

**补充笔记2：**
*   这些词都源于 `bind` (绑)。`bond` 是绑的东西；`bunch` 是绑起来的一束；`boundary` 是把一个区域“绑”起来的线；`border` 是一个区域的边缘。

**单词3：band / bandage / bend**
*   **band** 带子
*   **bandage** 绷带
*   **bend** 弯腰

**补充笔记3：**
*   `band` 和 `bandage` 都是扁平的带状物，用于捆绑。
*   `bend` (弯腰) 的形态就像一个被弯折的 `band` (带子)。
