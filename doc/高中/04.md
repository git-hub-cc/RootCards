**补充笔记1**
1.  **通假原理**：这组辅音字母之所以可以相互通假，是因为它们的发音非常相似。
2.  **D / T**：是一对清浊辅音（/d/ 和 /t/），发音位置相同，经常可以互换。
3.  **S / C**：字母 C 在很多单词中发 /s/ 的音（如 city, center），因此可以和 S 相互通假。
4.  **S / Z**：是一对清浊辅音（/s/ 和 /z/），发音位置相同，经常可以互换。
5.  **S / TH**：TH 发咬舌音 /θ/，很多学习者容易和 /s/ 混淆（例如 "thank you" 读成 "sank you"），说明它们的发音很接近，因此可以通假。
6.  **D-T-S 的关联**：这三个字母发音非常干脆，是人类发音难度最低的音之一，因此在单词演变和构词中非常活跃，经常互相转换。
