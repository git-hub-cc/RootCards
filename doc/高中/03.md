**词根/规律：辅音通假规律三 (C, K, G, Q, H, X)**
**补充笔记：** 这一组辅音在发音时都与喉咙后部有关，被称为“喉音”，因此可以相互通假。`x` 的发音 /ks/ 中包含了 /k/，所以也归入此类。

**单词77：think** 翻译：想
**单词78：thank** 翻译：感谢
**补充笔记：** `i` 和 `a` 元音通假。这两个词的关联在于：当你“想”起别人对你的好时，你就会去“感谢”他。先有 `think`（想），后有 `thank`（感谢）。

**单词79：ache** 翻译：疼痛
**单词80：agony** 翻译：极度痛苦
**补充笔记：** `ch` 和 `g` 相互通假，`ony` 是名词后缀。`ache` 是一般的疼痛，而 `agony` 是指精神上或肉体上极度的、难以忍受的痛苦。

**单词81：anchor** 翻译：锚；主持人
**单词82：angle** 翻译：角度；钩子
**补充笔记：** `ch` 和 `g` 相互通假。`anchor` (锚) 的形状是一个钩子，`angle` (钩子) 也是。`angle` 从“钩子”引申出“角度”的含义，因为角度就像一个钩子的形状。主持人被称为 `anchor` 是因为他在一档节目中起到像船锚一样稳定局面的核心作用。

**单词83：anxious** 翻译：焦虑的
**单词84：anger** 翻译：愤怒
**补充笔记：** `x` (/ks/) 和 `g` 相互通假。`anxious` (焦虑) 和 `anger` (愤怒) 在情绪上是递进关系，长时间的焦虑可能会升级为愤怒。

**词根/规律：辅音通假规律四 (D, T, S, TH, Z)**
**补充笔记：** 这一组辅音在发音时都与牙齿和舌尖有关，被称为“齿音”，因此可以相互通假。

**单词85：hide** 翻译：隐藏
**单词86：hat** 翻译：帽子
**补充笔记：** `d` 和 `t` 相互通假。古人发明帽子的初衷是为了“隐藏”自己的头部，抵御恶劣天气或遮挡阳光。

**单词87：shadow** 翻译：影子
**单词88：shade** 翻译：阴凉处
**补充笔记：** `dow` 和 `de` 发生通假。影子和阴凉处的意境非常相似，都是光线被遮挡后形成的暗处。

**单词89：sad** 翻译：悲伤的
**单词90：satisfy** 翻译：使...满意
**补充笔记：** `d` 和 `t` 相互通假。`satisfy` = `sat` (词根, 来自sad) + `-isfy` (动词后缀)。它的逻辑是：通过“满足”别人的需求，来消除他的“悲伤”。

**单词91：seed** 翻译：种子
**单词92：sow** 翻译：播种
**补充笔记：** `d` 和 `w` 发生通假，这是一种特殊情况。先有播种(sow)这个动作，才有种子(seed)这种东西。`o` 在 `e` 之前，所以顺序是 `sow → seed`。

**单词93：saw** 翻译：锯
**补充笔记：** `saw` 和 `sow` 都是动作，发音也相近。古人在“播种”时需要用工具在土地上划开一道口子，这个动作和“锯”东西的动作很像，都是来回划动的动作，因此它们是同源的。
**衍生词：see-saw** 翻译：跷跷板。`see` 是 `saw` 的变体，跷跷板上下运动的意境和锯东西来回的动作相似。

**词根/规律：辅音通假规律五 (L, R, N)**

**单词94：moral** 翻译：道德的
**单词95：normal** 翻译：正常的
**补充笔记：** `m` 和 `n` 相互通假。`moral` (道德的) 和 `normal` (正常的) 在古人看来是等同的。符合社会道德规范的行为就是正常的，不符合的就是不正常的。

**单词96：journey** 翻译：旅程
**单词97：journal** 翻译：期刊，日志
**补充笔记：** `ey` 和 `al` 作为后缀发生变化。`journey` 是长途旅行，而 `journal` 记录旅行过程中的日志或报告，因此同源。
**衍生词：**
*   **journalist** (记者): 在古代，记者需要进行长途旅行(journey)去采访。
*   **journalism** (新闻业): 记者所在的行业。

**单词98：wine** 翻译：葡萄酒
**单词99：vine** 翻译：葡萄藤
**补充笔记：** `w` 和 `v` 相互通假。`wine` (葡萄酒) 是从 `vine` (葡萄藤) 上结出的果实酿造的。
**衍生词：**
*   **vinegar** (醋): `vine` (葡萄酒) + `gar` (词根，表示酸)。放酸了的葡萄酒就是醋。
*   **vineyard** (葡萄园): `vine` (葡萄藤) + `yard` (院子)。种满葡萄藤的院子就是葡萄园。

**单词100：worm** 翻译：虫
**单词101：vermin** 翻译：害虫
**补充笔记：** `w` 和 `v` 相互通假。`vermin` 是指对人类有害的虫类。

**单词102：wall** 翻译：墙
**单词103：vault** 翻译：拱顶；金库
**补充笔记：** `w` 和 `v` 相互通假。`vault` (拱顶) 指的是像墙一样向上弯曲延伸形成的屋顶。而银行的“金库”之所以叫 `vault`，是因为金库的墙壁和屋顶都非常厚实，像一个坚固的拱顶结构。

**单词104：well** 翻译：好
**单词105：valid** 翻译：有效的
**补充笔记：** `w` 和 `v` 相互通假。一个东西是“好”的，它就是“有效”的。
**衍生词：**
*   **value** (价值): `val` (词根, 来自valid) + `-ue` (后缀)。有效的东西就是有价值的。
*   **available** (可获得的): `a-` + `vail` (词根, 来自valid) + `-able` (能...的)。能够获得的，就是对你有用的、有效的。

**单词106：want** 翻译：想要
**单词107：vanity** 翻译：虚荣
**补充笔记：** `w` 和 `v` 相互通假。`vanity` 的核心意境是“空虚”。人之所以会“想要”各种东西，是因为内心感到“空虚”。

**单词108：wind** 翻译：风
**单词109：ventilate** 翻译：使通风
**补充笔记：** `w` 和 `v` 相互通假。`ventilate` = `vent` (词根, 来自wind) + `-il-` (连接中缀) + `-ate` (动词后缀)。它的核心就是“风”，使一个地方有风，就是通风。
**衍生词：**
*   **vent** (通风口): 让风进出的口。
*   **ventilator** (通风设备): 用于通风的机器。
