**词根**
pact (打包)

**单词1** pack 打包, 包裹
**补充笔记1**
动词是打包，名词是包裹。

**单词2** package 包裹
**例句2** This is your package.
**补充笔记2**
`pack` 的名词形式，常指快递包裹。

**单词3** compact 紧凑的, 紧密的
**例句3** compact cars
**补充笔记3**
词根解析：`com-` (共同, 一起) + `pact` (打包)。把所有东西都打包在一起，画面感是挤压、塞满，因此意为“紧凑的、紧密的”。

**单词4** compatible 兼容的, 能和睦相处的
**例句4** Mildred and I are very compatible. / be compatible with sb/sth
**补充笔记4**
记忆方法：能够一起“拍”的，指能够合拍、兼容。

**词根**
tach, tag, tact (接触)

**单词1** touch 接触
**补充笔记1**
基础词。

**单词2** attach 贴上, 附上
**例句2** We attach labels to things before we file them away.
**补充笔记2**
词根解析：`at-` (to, 去) + `tach` (接触)。去接触，即将一个东西贴到另一个东西上。

**单词3** file away 归档
**补充笔记3**
`file` 在这里作动词，`away` 表示“放好、收起来”。`file them away` 指把它们作为文件归档放好。

**单词4** detach 分开, 拆下
**例句4** It is not easy to detach each link of the chain.
**补充笔记4**
词根解析：`de-` (off, 分开) + `tach` (接触)。从接触的状态中分离开。

**词根**
tag, tact (接触)

**单词1** contagious 传染性的
**例句1** a highly contagious disease of the lungs
**补充笔记1**
词根解析：`con-` (共同, 一起) + `tag` (接触) + `-ious` (形容词后缀)。很多人在一起互相接触导致的，即“传染性的”。

**单词2** contaminate 污染
**例句2** The river was contaminated by the waste water from the factory.
**补充笔记2**
词根解析：`con-` (共同, 一起) + `tam` (touch, 接触) + `-inate` (动词后缀)。不好的东西和好的东西接触在一起，导致“污染”。

**单词3** contamination 污染 (名词)
**补充笔记3**
