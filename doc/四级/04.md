### 笔记1：词根RAD, SAT, SEQ, SERT.md

**词根: RAD (词根=root 根)**
*   **补充笔记:** 词根RAD由root(根)演变而来，其核心画面感为像树根一样“发散、散射状”的形状。

**单词1: eradicate 根除，消灭**
*   **例句:** This problem has been eradicated. (这个问题已经被根除了。)
*   **补充笔记:** 词汇拆解：e-(out 向外) + rad(root 根) + -icate(后缀)。字面意思为“根都向外了”，引申为“斩草除根”。名词形式为eradication。

**单词2: radiate 发出，辐射，流露**
*   **例句:** People are sitting around a stove that radiates warmth. (人们围坐在一个散发着温暖的炉子旁。)
*   **补充笔记:** 词汇拆解：rad(root 根) + -i-(连接成分) + -ate(动词后缀)。动作像根一样发散，即发出、辐射。名词形式为radiation。

**单词3: radical 激进分子；根本的**
*   **例句:** There were radical differences between the two organizations. (这两个组织之间存在根本性的差异。)
*   **补充笔记:** 词汇拆解：rad(root 根) + -ical(后缀)。“激进分子”指的就是想从“根”上彻底改变事物的人。作为形容词时，意为“根本的”。

**单词4: radioactive 放射性的**
*   **例句:** The substance contains radioactive particles. (这个物质含有放射性粒子。)
*   **补充笔记:** 词汇拆解：radio(辐射) + active(活跃的)。放射性的物质每时每刻都在活跃地向四周辐射能量或粒子，因此是“活跃地辐射”。

**词根: SAT (词根=enough 满)**
*   **补充笔记:** 词根SAT源自Saturday(星期六)，在一些文化中，星期六是一周的最后一天，代表“满了”，引申为足够、饱和。

**单词1: saturate 浸透，使湿透，使饱和**
*   **例句:** The continuous rain had saturated the soil. (连绵的阴雨浸透了土壤。)
*   **补充笔记:** 词汇拆解：sat(满) + -ate(动词后缀)。使…充满，即浸透、饱和。关键在于理解“湿透”就是“满了，再也无法吸收水分了”。名词形式为saturation。

**词根: SEQ / SEC / SEQU (词根=follow 跟随)**
*   **补充笔记:** 该词根的核心画面感为一系列事物一个接一个地跟随，像一串连续的箭头。

**单词1: consecutive 连续的，连贯的**
*   **例句:** It has been raining for five consecutive days. (已经连续下了五天雨了。)
*   **补充笔记:** 词汇拆解：con-(共同) + sec(跟随) + -utive(后缀)。字面意为“全部一个一个跟着来”，即连续的。补充：“连续五天”也可表达为 `five days in a row`。

**单词2: sequence 连续，一连串，顺序**
*   **例句:** He described the sequence of the events leading up to the robbery. (他描述了导致抢劫发生的一系列事件。)
*   **补充笔记:** 词汇拆解：sequ(跟随) + -ence(名词后缀)。指一连串跟随的事物，即顺序、序列。

**单词3: subsequent 随后的，后来的**
*   **例句:** Subsequent investigations did not uncover any new evidence. (随后的调查没有发现任何新证据。)
*   **补充笔记:** 词汇拆解：sub-(在...下面/之后) + sequ(跟随) + -ent(后缀)。意为“在后面跟着的”，即随后的。副词形式为subsequently。

**词根: SERT (词根=put, join 放置，加入)**

**单词1: desert / desert 抛弃，遗弃 / 沙漠**
*   **例句:** She was deserted in the desert. (她被遗弃在沙漠中。)
*   **补充笔记:** de-(away 离开) + sert(放置)。把…丢在一边，即抛弃。沙漠是鸟不拉屎、寸草不生的地方，就像被上帝遗弃的地方，因此得名desert。注意重音区别：动词 `de-SERT`，名词 `DE-sert`。

**单词2: insert 插入，嵌入**
*   **例句:** Insert coins into the slot. (将硬币插入投币口。)
*   **补充笔记:** 词汇拆解：in-(in 往里) + sert(放置)。向里面放置，即插入。
